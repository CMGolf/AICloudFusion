# AI Cloud Fusion — Lab Guides

Welcome to the AI Cloud Fusion workshop lab guides. These hands-on labs accompany the weekly workshop sessions and are designed to be completed independently using only the instructions in each guide.

## How to Use These Labs

Each session has its own folder containing multiple labs at different difficulty levels:

- **Lab A (Beginner)** — Foundational setup or introductory concepts
- **Lab B (Intermediate)** — Core hands-on task with multiple AWS services
- **Lab C (Advanced)** — Real-world engineering task that builds on the session's concepts

Work through the labs in order (A → B → C). Each lab includes:

- Step-by-step instructions with exact commands
- Annotations explaining what each service does and why
- Console checkpoints to verify your work visually
- Cleanup/decommission steps so you don't incur any costs

## Prerequisites

- A free-tier AWS account ([Create one here](https://aws.amazon.com/free/))
- Access to AWS IAM Identity Center (provided by your facilitator)
- A computer running Windows, macOS, or Linux

## Cost

All labs are designed to stay within the AWS Free Tier. Each lab includes a cost notice listing the services used and their free tier limits. **Always complete the Cleanup section at the end of each lab** to ensure no resources are left running.

## Need Help?

If you get stuck on any lab, post your question in the **Lab Help** channel on Microsoft Teams. Include the command you ran and the error message you received.
